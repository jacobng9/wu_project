## Scripts

| Script | What it does |
|---|---|
| `Processing Pipeline Samples.R` | Upstream pipeline — loads raw 10x h5 files, QC, doublet removal, DecontX, Harmony. Run on the lab server. |
| `Cell Type Annotations.R` | Broad cell type annotation (Colonocytes, T cells, B cells, etc.) + colonocyte sub-annotation |
| `Processing(Mine).R` | Goblet sub-annotation starting from the **full 15.7 GB** RDS. Requires a large-memory machine. |
| `Goblet_only_jacob.R` | **Use this one.** Same plots as `Processing(Mine).R` but loads the **4.7 GB goblet-only** RDS. Runs locally. |
| `unified_plots.R` | Colleague's (Aayush's) combined plot script — update `root_path` to your machine before running. |
| `Two Way ANOVA and DE Analysis.R` | Differential expression and two-way ANOVA across Genotype × Diet conditions |
